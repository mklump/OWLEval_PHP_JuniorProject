Use Case: Create custom question forms

 

Description: 

This Use Case describes the process when an instructor want to create a form with customized questions. 

 

Actor(s): Instructor, Staff (secondary)

 

Preconditions:

Question Number one must be included on any form

 

Basic Path: 

This Use Case is initiated when:

  1. Actor logs in to OwlEval
  2. Actor opens form template from menu
  3. customized questions are added
  4. validation of custom questions by Admin staff
  5. form is include in evaluation set
  6. system is notified

 

Alternate Paths:

1.     Instructor request staff to completes new questionnaire

 

Exception Paths:

4. question not excepted or question #1 excluded

4.1 Instructor notified

4.2 request returned

4.3 new request is submitted

4.4Return to step 3 of Basic path

 

Post conditions:

New Course evaluation question form is added to set

 

Related Data:

All tables and reports in course evaluation set

 

Business Rules:

Acceptable question set and question #1 included

 

Requirements:

This Process must be completed prior to distribution of evaluations
