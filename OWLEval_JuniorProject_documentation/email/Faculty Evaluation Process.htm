Listed below are the steps I currently go through each term to create, process and print results for the faculty evaluation process.  I do this process twice each term because the classes taught at Boeing have their data kept separate from everyone else.  All Portland sites and Klamath Falls are combined though.  Is it important for you to know that besides Metro and CAPITAL center, we also have two other satellite programs evaluated each term that are taught with OHSU instructors at two different locations?
 
Important term to know first:
SIS - Student Information System - This is the OIT campus program where all data about OIT students and classes are kept and updated.
 
Steps:
Create Evaluation Set Tables — This is done on the Monday following the "last day to drop with a W."  It takes the data from SIS at that point and puts it in a table.  It excludes some types of classes such as Independent Study, Seminars, Distance Ed, etc.
 
Merge Evaluation Sets (If classes are added after initial run) —  If an instructor wants to include a class that is usually excluded, they are allowed to do so in most cases.  I then have do step one again and create an evaluation set for just those added classes and then perform this step of merging the two sets.
 
Load an Evaluation Set Table for Processing — I choose which set to work with in this step.  I will have a Boeing set and and Regular set active at the same time, so I have to let the program know which one to use.  It stays on that set unless I come back to this step and change to the other one.
 
Evaluation Course Tracking List — This step prints a hard copy of all the classes in the set.  It includes the CRN, instructor, mailstop, etc.  I use it to check off when packets are returned.  It may not be needed for an online process.
 
Distribution Labels (8x11) — This step prints an 8x11 label to place on the outside of the packet.  It contains CRN, class, instructor, location, number of students, etc.  This step will not be needed for an online process.
 
Blue and Manila Envelope Labels — These steps print out the small label that is placed on each manila and each blue envelope.  It contains the CRN, class, instructor, location, number of students.   This step will not be needed for an online process.
 
Create custom question forms — Instructors are allowed to modify their forms to have different questions except question number one has to remain.  I use a separate program to create and print these forms.  They can even have different questions for different classes.
 
Packets are stuffed with forms — All the labels printed above are attached to the appropriate envelopes.  Forms are put into each envelope according to the class count listed on the label.  The out of town packets are mailed and the on campus ones are left in each building at the common secretary's desk area in locked boxes.  The actual filling out of forms happens the last two weeks before finals week.  This step is totally a manual process, so it definitely won't be needed for an online process and will save the most time.
 
Load Results File from F: drive — After all the packets have been returned, which is the day finals start, I use a scanner machine to get the results into a notepad file.  It is just a regular text file.  I then place these files on the network drive F in a certain folder.  There are three files: one with the standard forms data, one with custom forms data, and one with the custom questions for each instructor that choose to use them
 
Calculate Evaluation Results — This step calculates each instructors results.
 
Faculty Evaluations Instructor Reports- Recap and Details — This step prints the reports that go back to the instructor.  The recap just shows for all classes that person taught what their average was on question one.  The details report shows for each class how many students answered each question with each answer.  Reports can not be returned to the faculty until after the date that all grades must be turned in, which is the Tuesday after finals usually.  Usually it takes a little while longer anyway and they get their reports shortly after returning for the next term.
 
White Envelope Return Labels  — This step just prints labels to put on the envelopes we use to send the faculty their reports.  So this is another step that will be eliminated if it is online.
 
Calculate Institutional Totals  —  After the instructors have had about one week to look over their reports for problems, this step is done.  It calculates the totals across all the campuses for just question number one.
 
Faculty Evaluations Administration Reports - College Recap, Department Recap, Campus Recap, Class Type Recap, and Instructor Rank Recap  —- These steps create the reports that are given to the President, Provost and some are given to the Department chairs.  The Department Chairs also get a copy of the Instructor reports that were given to the individual instructors.
 
The blue comment forms are not opened by anyone but the instructor themselves here in Klamath Falls.  I believe in Portland that the someone types up the responses???  This is something that I'm not involved with, so you'll need to check with the ladies in Portland about what they do.
 
The supported web browser for OIT including the general OIT webpage and Web for Student is Internet Explorer, so I'd recommend creating your program to work well on it. 
 
At this time, students don't have access to these reports that I'm aware of.  I don't know if they asked, if they could see an instructor's results or not.  I keep copies in case someone loses their printed copy.  The Provost Office could let you know if students would be allowed access to reports and which ones.
 
So this gives you the overview of what I do for steps.  Let me know if you need an explanation on something.
