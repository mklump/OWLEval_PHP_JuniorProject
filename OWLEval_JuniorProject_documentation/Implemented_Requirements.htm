OwlEval Alpha Release

Implemented Project Requirements

Requirements

 

1) Compatible with OIT’s current hardware, software, and process.

           

2) Web Based Interface

            2.1) Shall work correctly with at least one web browser available at OIT.

            2.2) Shall separate interface based on user role.

 

3) Secure

            3.1) Shall Identify students only with randomly generated access key and password.

            3.2) Shall restrict data input to access keys identified with a particular course.

            3.4) Shall restrict access to database.

            3.5) Shall restrict access to program based on Role.

 

4) Collects, Displays, and Processes Evaluation Data

            4.2) Shall store Course information and data collected from students.

            4.3) Shall not lose data.

 

5) User-Configurable

           

6) Maintenance Free

 

7) System has Role-based Availablity

            7.1) Administrators shall have access to system at all times.

 

8) Testing

            8.1) Shall be able to handle multiple simultaneous submissions.

            8.2) Shall accept all printable characters.
