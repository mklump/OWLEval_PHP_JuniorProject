Instructor Use Case Psuedocode

 

NON-STUDENT LOGON

 

Log on web page is displayed

 

Logon information is entered then the submit button is clicked or the enter key is pressed.

 

The web page sends the logon information to the validation module on the web server.

 

The validation module processes the logon information.

 

If the logon information is authenticated the instructor’s main page or the Admin’s main page is loaded up in the browser and a session ID is issued. Otherwise an error message is displayed.

 

INSTRUCTOR’S MAIN PAGE

 

Instructor’s main page is displayed showing all the instructor’s courses.

 

The instructor chooses a course by clicking on a link or button.

 

The web page sends the course information and session ID to the reports module of the web server.

 

The reports module will validate the session ID and check to see if the evaluation for the class is ready. Before the evaluation is issued to students the add/modify questions web page is loaded up. If the evaluation has been issued to students but has not been completed then a message is displayed with a date to return for the results (the main page is loaded up after a button is pressed). If the evaluation has been completed then the reports module will submit a report request to the database SQL server.

 

ADD/MODIFY QUESTIONS

 

Instructor’s add/modify questions page is displayed showing all the questions, buttons to add or modify a question or to go back to the main page, and a drop down menu for all the courses that the instructor teaches.

 

The instructor selects either to modify an existing question or to add a new question.

 

If the add question was selected then the empty question web page is loaded. Otherwise the question web page is populated with the question to be modified.

 

The instructor enters/modifies the question and selects what course it is for.

 

The instructor submits the question for admin review and acceptance.

 

The web page submits the question to the pending question approval module on the web server.

 

The instructor’s add/modify questions page is displayed again.

 

EVALUATION REPORT

 

The database SQL server returns the evaluation report to the report module.

 

The report is displayed on the web page.

 

The instructor will read and/or print the evaluation then when finished will click on the finished button to return to the instructor’s main page.
