Non-Student Login
Maps to Requirments 1, 2, 6, 7, and 8

  1. Go to URL http://www.culbertson.info/blues/gwr/admin_logon.php and login with user name guest and password guest. Any other user names (except for the blues team logons) or passwords will not work.

Create or Add Questions to the Evaluation
Maps to Requirments 1, 2, 5.2, 7, and 8

  1. From the MENU click on the Add/Remove Question button.
  2. Select the CRN that you want to view the results for. It might already be selected (showing in the selection box).
  3. Click the Augment button.
  4. Type in a question you want the students in your class to answer in the box and click the Save All Changes button.
  5. You will see a message at the bottom that the changes were successful.
  6. The question you just entered can be modified by typing or it can be removed from the system by clicking the Remove button.
  7. You may enter a new question if desired in the bottom box.

Create and Print Evaluation Keys for Student Login
Maps to Requirments 1, 2, 3.1, 3.2, 7, and 8

  1. From the MENU click on the Create Evaluation button.
  2. Click on the Create Keys button and the Eval Management page will be displayed.
  3. Select the name guest from the Instructor Name field.
  4. Enter a five digit number into the CRN field. Any other number of digits or any non numeric character will fail.
  5. Enter a positive non-zero number into the Number of students in the course field. Any negative number or any non numeric character will fail.
  6. Enter a Valid date and time format into the When do you want the evaluation to start? field. Any invalid date or time or a date and time before now will fail. You can use the word now if you want.
  7. Enter a Valid date and time format into the When do you want the evaluation to end? field. Any invalid date or time will fail. If the end date and time is not after the start date and time it will fail. You can use the word tomorrow if you want.
  8. Click on the Create Keys button.
  9. If an error was made it will be displayed. Click on the Back button to go back and enter the information correctly.
  10. When the correct information is entered the keys will be created, entered into the database, and displayed on the web page. Use the following URL to check the status of the database http://www.culbertson.info/blues/glenn/show_db.php. This will show the student keys that were just generated under the Student Keys section for use below. These keys will now allow the student to log in.
  11. There will be three buttons displayed; Print keys, Delete keys, and Back buttons. 
  12. Click on the Print keys button. 
    • The print dialog box will automatically open. 
    • Move the print dialog box out of the way as needed so you can read the directions at the top of the web page by clicking the top bar and dragging it to the desired location.
    • Read the top of the web page and set up the font and margins as specified.
    • From the print dialog box select a printer then click Print.
    • Use the browsers back button (not the one at the bottom of the page) to go back to the previous page (Create Keys Part 2) so you can test the other two buttons later.

Student Login Page
Maps to Requirments 1, 2, 3.1 to 3.5, 7, and 8

  1. Open a new browser window then go to URL http://www.culbertson.info/blues/glenn/student_login.php
  2. Page should load with all images and controls in tact. Cursor
    should be defaulted to key text-box.
  3. Login with a key that was generated above. If you experience any problems with the newly created key you may use the word apple as a key. Any other key will not work.
    1. Entering a Null key will show an error for no entry and the evaluation will not be displayed.
    2. Entering an  invalid key will show an error for a bad key and the evaluation will not be displayed.
  4. Click on Begin Evaluation button.
  5. Evaluation page should load with any added questions last.

Evaluation Page Student View
Maps to Requirments 1, 2, 3.2, 3.4, 3.5, 4.2, 4.3, 7, and 8

  1. Click the appropriate option buttons and enter answers to the questions on the form.
  2. Press Save Evaluation button.
  3. A "thank you" message is displayed.
  4. The key that the student used (other than the apple key) will be deleted from the database.
  5. Use the following URL to check the status of the database http://www.culbertson.info/blues/glenn/show_db.php. 

View Evaluation
Maps to Requirments 1, 2, 3.2, 3.4, 3.5, 4.1, 4.4, 7, and 8

  1. Go to URL http://www.culbertson.info/blues/gwr/admin_logon.php and login with user name guest and password guest.
  2. From the MENU click on the View Evaluation Results button.
  3. Select the CRN that you want to view the results for. It might already be selected (showing in the selection box).
  4. Click the View button.
  5. View the results.

Delete Evaluation Keys
Maps to Requirments 1, 2, 3, 7, and 8

  1. Do one (or both) of the following
  • Go to the other browser window that you logged in with the non-student log in and click on the Delete keys button

or

  • go to URL http://www.culbertson.info/blues/gwr/admin_logon.php and do the following.
    • Login with user name guest and password guest.
    • From the MENU click on the Create Evaluation button.
    • In the left box, select the CRN that was entered above to delete the keys that were created.
    • Click on the Delete keys for CRN button.
  1. The status of deleting the keys is displayed.
  2. There will be three buttons displayed.
  • The Delete Results button will delete all student entries for the CRN displayed on the page. This button can be ignored at this time since there aren't any results to delete. If you want to click on it you may but use the browsers back button to return to the erase.php page.

  • Click on the manual button to make a single key entry into the database. 

    - Enter the CRN, key, time start and time end and click on the Insert Key button.
    - The results are displayed. Click on the Back button to go back to the single key entry page.

  • Click on the Back button to go back to the Eval Management page.

Edit an Evaluation
Maps to Requirments 1, 2, 3.4, 3.5, 3.8, 4.3, 5.2, 7, and 8

  1. From the Eval Management page click on the Edit Eval button on the top navigation bar.

  2. Select the CRN that you want to edit.

  3. Click on the Retrieve information for CRN above button.

  4. The page will display all information about the CRN. The desired information can be altered then click on the Update information button.

  5. The page will display the results of the update.

  6. You may either print the keys by clicking the Print keys button or go back by clicking the Back button to go back to the Edit Evaluation page.

  7. Click on the Back button to go back to the Eval Management page.

View/Save an Evaluation
Maps to Requirments 1, 2, 4.2, 4.3, 7, and 8

  1. From the Eval Management page right side box select the CRN that you want to view/save. Select 0 to view/save all information for all CRNs.

  2. Click on the View Eval button.

  3. The Save HTML Document dialog box will appear. Edit the name in the file name box, navigate to the desired location to save the eval, then click on the Save button.

  4. There is a button on the page that can be selected if the above is cancelled.

Navigation Button Bar
Maps to Requirments 1, 2, 7, and 8

  1. Click on any button on the top Navigation Button Bar to quickly navigate to the desired portion of the site.

  2. Click on the Admin Site Map button to display a graphic showing the links between the various pages.
